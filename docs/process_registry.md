# Process Registry

This document tracks all Process IDs (PIDs) and their associated governance artifacts.

## PID Assignment Rules
- PIDs follow format: `PID-XXX`
- Each PID maps to a specific business process or use case
- All PIDs must have corresponding documentation artifacts
- PIDs are reserved sequentially (001, 002, 003, etc.)
