Framework:
- MUST use vanilla JavaScript (no React, Vue, Angular, or other frameworks)
- Pure HTML, CSS, and vanilla JavaScript only

Deployment:
- Container: nginx:alpine
- Target subpath: /$app_name_kebab/
- MUST include nginx.conf for subpath routing configuration
- Nginx must strip /$app_name_kebab/ prefix for static assets
- Nginx must redirect root to /$app_name_kebab/
- Nginx must redirect /$app_name_kebab (without trailing slash) to /$app_name_kebab/

API Integration:
- Agent Status endpoint: http://localhost:8080/agents/status
- Health Check endpoint: http://localhost:8080/health
- Display real framework version from environment variables
- Show actual agent status from health APIs

Required Files (Minimum):
- MUST include nginx.conf (for deployment)
- Standard web app needs: HTML entry point, CSS, JavaScript

Output Format:
- HTML: Must start with <!DOCTYPE html>
- HTML: Use relative paths (no base href tags)
- No markdown code block markers in any file
- Include footer: "v$version | WarmBoot Run: $run_id"

Quality Standards:
